# Copywriting

When the user wants to write, rewrite, or improve marketing copy for any page — including homepage, landing pages, pricing pages, feature pages, about pages, or product pages. Also use when the user says "write copy for," "improve this copy," "rewrite this page," "marketing copy," "headline help," or "CTA copy." For email copy, see email-sequence. For popup copy, see popup-cro.

## What it is
An agent-agnostic marketing skill focused on **copywriting**.

## When to use it
Use this skill in the following scenarios:
- When the user requests: *"write copy"*
- When the user requests: *"marketing copy"*
- When the user requests: *"headline"*
- When the user requests: *"CTA copy"*
- When the user requests: *"landing page copy"*
- When the user requests: *"homepage copy"*
- When the user requests: *"rewrite"*
- When the user requests: *"improve copy"*

## How to use
1. Trigger this skill by asking the marketing agent for tasks related to **copywriting**.
2. The agent will read `SKILL.md` to guide its execution workflow.
3. Ensure you provide relevant product positioning and customer context, which can be generated using the `clarify-product` skill.
