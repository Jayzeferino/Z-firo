# Brand Building

Brand strategy, identity, positioning, and voice development. Use when developing brand guidelines, creating positioning statements, defining brand voice, or building brand architecture.

## What it is
An agent-agnostic marketing skill focused on **brand building**.

## When to use it
Use this skill in the following scenarios:
- When the user requests: *"brand"*
- When the user requests: *"branding"*
- When the user requests: *"brand voice"*
- When the user requests: *"positioning"*
- When the user requests: *"brand guidelines"*
- When the user requests: *"brand identity"*
- When the user requests: *"brand strategy"*
- When the user requests: *"tone of voice"*

## How to use
1. Trigger this skill by asking the marketing agent for tasks related to **brand building**.
2. The agent will read `SKILL.md` to guide its execution workflow.
3. Ensure you provide relevant product positioning and customer context, which can be generated using the `clarify-product` skill.
